*************************
Instagram Scraper
*************************

The main instructions of the IG Follower Scraper are inside the Intructions.txt file. To ensure that the program 
runs smoothly, please make sure to follow those instructions exactly. 

Also, the way that the git repository has been set up is the way that the files should be arranged once you've 
downloaded them. If you download the entire repository as is, you shouldn't have a problem running the program. 

The IG Followe Scraper will ask you whose followers you want to download. 

You supply the username of the account that you want to scrape. If their followers are available for public 
viewing, then you can download up to 20,000 followers at a time. 

To prevent your account from being blocked, the API calls will only download 1,000 followers at a time before 
taking a break and continuing. This is why the limit is 20,000. 

You can run the program more than once if you need more than that. 

This program is meant to go hand in hand with the IG-Messenger-Bot that is also available in my repository at: 
https://github.com/ArmanTheAutomator
